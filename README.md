# n8n-Send-promo-code-expiry-alerts-using-Webhook-Google-Sheets-Gmail-and-Slack
Automated Promo Code Expiry Alert System built with n8n, Webhooks, Google Sheets, Gmail, and Slack. Tracks promo code expiry, stores records, calculates urgency, and sends real-time email and Slack alerts. Ideal for eCommerce, marketing campaigns, and discount management.
